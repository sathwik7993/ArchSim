# ArchSim Keyboard Shortcuts

This document outlines the keyboard shortcuts supported by the ArchSim client to optimize workflow efficiency for power users.

---

## 1. Playback & Simulation Controls

| Keystroke | Action | Context |
| :--- | :--- | :--- |
| `Space` | Play / Pause Simulation | Global |
| `S` | Step Forward (1 Virtual Tick) | Paused Mode |
| `R` | Reset Simulation Timeline to 00:00:00 | Global |
| `]` | Double Simulation Speed | Global |
| `[` | Halve Simulation Speed | Global |

---

## 2. Canvas & Node Interactions

| Keystroke | Action | Context |
| :--- | :--- | :--- |
| `V` / `Escape` | Switch to Select Tool | Canvas Active |
| `C` | Switch to Connector Tool (Link creation mode) | Canvas Active |
| `Delete` / `Backspace`| Delete selected components or links | Selection Active |
| `Ctrl + Z` | Undo last canvas operation | Global |
| `Ctrl + Y` | Redo last canvas operation | Global |
| `Ctrl + A` | Select all components on canvas | Canvas Active |
| `Ctrl + D` | Duplicate selected nodes | Selection Active |
| `Ctrl + G` | Group selected nodes into a visual box | Multiple Nodes Selected|
| `F` | Zoom to Fit (Centers viewport on elements) | Canvas Active |

---

## 3. Panels & Sidebar Toggles

| Keystroke | Action | Context |
| :--- | :--- | :--- |
| `Ctrl + B` | Toggle Right Inspector Panel visibility | Global |
| `Ctrl + T` | Toggle Traffic Generator Panel visibility | Global |
| `Ctrl + H` | Toggle bottom Timeline / Chaos panel | Global |
| `?` | Show / Hide Keyboard Shortcuts Modal | Global |
